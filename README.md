# Sorting algorithms
This repository contains:
1. Bubble Sort
2. Selection Sort

## Bubble Sort
Bubble sort is a simple sorting algorithm

### Algorithm
1. Compare adjacent elements.
2. Swap if they are in wrong order
3. Repeat until sorted.

### Time Complexity
- Best: O(n)
- Average: O(n^2)
- Worst: O(n^2)

### Selection sort
Selection sort is a simple comparision based sorting algorithm.
It repeatedly selects the smallest element from the unsorted portion of the array and places it at the beginning.

## Algorithm
1. Start from the first element of the array.
2. Assume the first element is the minimum.
3. Compare this minimum element with the rest of the elements.
4. If a smaller element is found, update the minimum.
5. Swap the minimum element with thw first element of the unsorted part.
6. Move to the next position and repeat until the array is sorted.

## Time Complexity
- Best: O(n^2)
- Average: O(n^2)
- Worst: O(n^2)

- v2.1 - User input support
